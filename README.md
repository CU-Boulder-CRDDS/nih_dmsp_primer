# Navigating the NIH Data Management and Sharing Policy (DMSP): Tools for CU Researchers

This repository contains materials associated with CRDDS's primer on  NIH's 2023 Data Management and Sharing Policy (DMSP). [View the website](https://cu-boulder-crdds.github.io/nih_dmsp_primer/).

As of 2026-05-26, NIH is using a format for DMSPs. Please refer to [NIH's website](https://grants.nih.gov/grants-process/write-application/forms-directory/data-management-and-sharing-plan-format-page) for information on their 2026 Pilot DMS Plan format while we work on updating our documentation.
